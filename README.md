This is a collection of some real-world projects available on DataCamp that can be carried out using Python.

# 1 Analyzing Crime in Los Angeles
**Description**: You have been asked to support the Los Angeles Police Department (LAPD) by analyzing crime data to identify patterns in criminal behavior. They plan to use your insights to allocate resources effectively to tackle various crimes in different areas.

**Dataset**:The dataset is a modified version of the original data, which is publicly available from Los Angeles Open Data.

**Questions to answer** 
1. Which hour has the highest frequency of crimes?
2. Which area has the largest frequency of night crimes (crimes committed between 10pm and 3:59am)?
3. Identify the number of crimes committed against victims of different age groups.


# 2 Exploring NYC Public School Test Result Scores
**Description**: You have been tasked with answering three key questions about New York City (NYC) public school SAT performance. SATs are standardized tests intended to measure literacy, numeracy, and writing skills.

**Dataset**:The dataset is a modified version of the original data, which is publicly available from NYC Open Data.

**Questions to answer** 
1. Which NYC schools have the best math results? 
2. What are the top 10 performing schools based on the combined SAT scores? 
3. Which single borough has the largest standard deviation in the combined SAT score?
